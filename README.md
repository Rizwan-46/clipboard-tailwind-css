# 📋 Clipboard Landing Page

A modern, fully responsive landing page for a cross-platform clipboard synchronization application. This project demonstrates responsive web design using **HTML5** and **Tailwind CSS**, with a focus on clean layouts, reusable utility classes, and an engaging user experience.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Rizwan-46/clipboard-tailwind-css.git

cd clipboard-tailwind-css
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Tailwind CSS Watch Mode

```bash
npx tailwindcss -i ./input.css -o ./output.css --watch
```

If your `package.json` includes custom scripts, you can also run:

```bash
npm run dev
```

or

```bash
npm run watch
```

### 4. Open the Project

Open `index.html` directly in your browser or use the **Live Server** extension in Visual Studio Code for automatic reloading during development.
